# TechTrek_CE3

~ For Running the Project on server :

Flask Project:
	1. Create and Activate the virtual environment.
	2. Install the required libraries:
	  pip install flask flask-login flask-SQLAlchemy flask-jwt-extended flask_wtf email_validator flask-restful
	3. Run the Server:
		python app.py


Django Project:
	1. Create and Activate the virtual environment.
	2. Install the required libraries:
	 	pip install django requests djangorestframework
	3. Run the Server:
		python manage.py runserver
